## 六、发现的问题和解决方案

### 6.1 测试设计问题（已解决）

**问题1**: CheckInService测试设计缺陷
- **描述**: 测试期望第8次签到（重复签到）的bonus，但应该验证第7次签到的bonus
- **影响**: 测试失败
- **解决**: 修正测试代码，保存第7次签到的result并验证其bonus
- **责任人**: DEV Agent（修正明显错误的测试）

**问题2**: ShopService测试违反业务逻辑
- **描述**: 测试直接应用未购买的皮肤，违反PL文档定义的购买流程
- **影响**: 测试失败
- **解决**: 修正测试代码，先purchaseSkin()再applySkin()
- **责任人**: DEV Agent（修正明显错误的测试）
