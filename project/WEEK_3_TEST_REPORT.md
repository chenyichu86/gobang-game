## 八、问题和挑战

### 已解决的问题

1. **AI Client异步导入**
   - 问题: require()在ES模块中不可用
   - 解决: 使用动态import()

2. **Web Worker作用域**
   - 问题: triggerAIMove()无法访问get()
   - 解决: 将函数移到create()回调内部

3. **GameEngine缺少undo**
   - 问题: PVE模式需要悔棋功能
   - 解决: 在GameEngine中添加undo()方法

4. **测试异步等待**
   - 问题: AI落子需要时间，测试有时序问题
   - 解决: 使用setTimeout等待300ms确保AI完成

### 已知限制

1. **ai-client.ts覆盖率低**
   - 原因: Web Worker代码在浏览器环境运行
   - 影响: 测试环境使用同步实现，覆盖率40%
   - 建议: 需要浏览器环境测试才能提高覆盖率

2. **AI难度有限**
   - SimpleAI: 80%随机+20%防守
   - MediumAI: 评分系统+棋型识别
   - 未实现: 困难AI和大师AI（Week 4+）
